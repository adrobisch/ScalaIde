# Scala IDE

_Requirements_:

  * [Typesafe Stack](http://typesafe.com/stack/download)
  * [Play Framework](http://www.playframework.org/documentation/2.0/Installing)

_Run it_:

`git clone git@github.com:themerius/ScalaIde.git`

`cd ScalaIde`

`play run`

_To get access to the databse_:

'play h2-browser run'


# Technologies

* Scala
* Play Framework
* HTML5
